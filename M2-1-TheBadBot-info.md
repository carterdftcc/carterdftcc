### **Fundamental Concepts and the CIA Triad**
The **Confidentiality**, **Integrity**, and **Availability** (**CIA**) triad represents the core security goals for information and systems.

* **Cybersecurity:** The practice of protecting systems, networks, and data from digital attacks.
* **Confidentiality:** The principle of ensuring data is only accessible to authorized individuals.
* **Integrity:** The principle of safeguarding the accuracy and completeness of data by preventing unauthorized modifications.
* **Availability:** The principle of ensuring systems and data are accessible and usable by authorized individuals when needed.
* **Threat:** A potential source of an attack or an adverse event.
* **Vulnerability:** A weakness in a system or its security procedures that can be exploited by a threat.
* **Exploit:** A piece of software, data, or a sequence of commands that takes advantage of a vulnerability.
* **Risk:** The potential for an unwanted outcome, measured by the likelihood of an event and its impact.
* **Attack Vector:** The path or method used by a hacker to gain unauthorized access to a system.
* **Zero-Day Exploit:** A newly discovered software vulnerability that has no existing patch and is actively being exploited.
* **Defense-in-Depth:** A cybersecurity strategy that uses multiple, overlapping security layers to protect an organization's assets.
* **Threat Actor** (or Agent): An individual, group, or organization that carries out malicious activities.
* **Insider Threat:** A security risk coming from within an organization, such as a malicious or careless employee.
* **Phishing:** An attack that uses deceptive emails or messages to trick people into revealing sensitive information.
* **Social Engineering:** The psychological manipulation of people into divulging confidential information or performing certain actions.

### **Malware and Malicious Attacks**
**Malware** is a general term for malicious software designed to disrupt or damage systems. Students will learn to differentiate between the various forms of malware and the attacks that deploy them.

* **Malware:** (Malicious Software) A broad term for any software designed to harm a computer system.
* **Virus:** A type of malware that attaches itself to other files and can spread to other computers by replicating itself.
* **Worm:** A self-replicating and self-propagating malware that spreads across networks without needing human intervention.
* **Trojan Horse:** A malicious program disguised as legitimate software to trick users into installing it.
* **Ransomware:** Malware that encrypts a victim's files and demands a ransom payment to restore access.
* **Adware:** Software that automatically displays or downloads unwanted advertising material.
* **Spyware:** Malware that secretly monitors and collects information about a user's activities.
* **Keylogger:** A type of spyware that records every keystroke a user makes.
* **Rootkit:** A malicious software designed to gain privileged access to a computer and hide its presence from the user.
* **Botnet:** A network of computers infected with malware and controlled by a remote attacker.
* **DDoS** (**Distributed Denial-of-Service**): An attack that overwhelms a server with a flood of traffic from many different sources.
* **Man-in-the-Middle (MitM) Attack:** An attack where an attacker secretly intercepts and relays messages between two parties.
* **Brute Force Attack:** A trial-and-error attack that tries all possible password combinations to gain access to a system.
* **Spoofing:** A cyberattack where a malicious party impersonates another device or user to launch attacks against a computer network.
* **Watering Hole Attack:** A strategic attack that infects a website frequently visited by a specific group of users.

### **Security Technologies and Processes**
These terms refer to the tools and practices used to protect systems and respond to incidents.

* **Firewall:** A network security system that monitors and controls incoming and outgoing network traffic.
* **Antivirus Software:** A program designed to detect, prevent, and remove malicious software.
* **VPN** (**Virtual Private Network**): A technology that creates a secure, encrypted connection over a public network, such as the internet.
* **Encryption:** The process of converting data into an unreadable code to prevent unauthorized access.
* **Decryption:** The process of converting encrypted data back into its original, readable format.
* **Cryptography:** The science and practice of securing communication by using codes to protect data.
* **Hash Function:** An algorithm that creates a unique, fixed-size string of characters from a variable-sized input.
* **Authentication:** The process of verifying the identity of a user or device before granting access.
* **Authorization:** The process of determining what resources an authenticated user is permitted to access.
* **Access Control:** The practice of restricting access to resources in a computing environment.
* **Multi-Factor Authentication (MFA):** A security method that requires a user to provide two or more verification factors to gain access.
* **Password Manager:** A software application that helps users store and organize their passwords securely.
* **Patch:** A software update designed to fix bugs or security vulnerabilities.
* **Patch Management:** The process of applying patches and updates to software and systems.
* **Incident Response Plan:** A step-by-step procedure that an organization follows when a security breach occurs.
* **Honeypot:** A decoy system designed to attract and trap attackers to monitor their activity.
* **Sandbox:** A security mechanism that runs untrusted programs in a safe, isolated environment.
* **Security Operations Center (SOC):** A centralized unit within an organization that manages security issues.
* **Penetration Testing:** A simulated cyber attack against a system to check for exploitable vulnerabilities.
* **Ethical Hacking:** The authorized practice of hacking to identify security vulnerabilities.
* **White Hat Hacker:** A cybersecurity expert who uses hacking skills for defensive and ethical purposes.
* **Black Hat Hacker:** A malicious hacker who exploits vulnerabilities for illegal or malicious purposes.
* **Red Team:** A group of people who act as attackers in a penetration test to find weaknesses in a system.
* **Blue Team:** A group of people who defend a system against a red team's attacks.
* **Purple Team:** A group that combines the efforts of both the red and blue teams to improve an organization's overall security.

### **Network and Data Concepts**
These terms cover the underlying infrastructure and information types that are protected in cybersecurity.

* **Network:** Two or more devices connected to exchange data.
* **IP Address:** A unique numerical label assigned to every device on a network.
* **Router:** A device that forwards data packets between computer networks.
* **Switch:** A network device that connects devices within a single network.
* **Packet:** A small unit of data transmitted over a network.
* **DNS** (**Domain Name System**): The system that translates domain names (like `example.com`) into IP addresses.
* **Port:** A communication endpoint in a computer's operating system.
* **Cloud Computing:** The delivery of computing services, including storage and processing power, over the internet.
* **Data Breach:** A security incident where sensitive, confidential, or protected data is copied, stolen, or exposed.
* **PII** (**Personally Identifiable Information**): Any information that can be used to identify, contact, or locate an individual.
* **Metadata:** Data that provides information about other data.
* **Internet of Things (IoT):** The network of physical devices with unique identifiers that are connected to the internet.

### **Web and Software Vulnerabilities**
Attackers often target web-based systems and applications.

* **SQL Injection (SQLi):** An attack that injects malicious code into a server that uses SQL.
* **Cross-Site Scripting (XSS):** An attack that injects malicious scripts into web pages.
* **Clickjacking:** An attack that tricks a user into clicking on a hidden, malicious link.
* **CSRF** (**Cross-Site Request Forgery**): An attack that tricks an authenticated user into unknowingly submitting a malicious request.
* **Web Browser:** An application used to browse the internet.
* **Cookie:** A small text file sent from a website to a web browser to record user activity.
* **Cache:** A temporary storage location for web pages and files to speed up browsing.

### **Compliance and Governance**
Cybersecurity is governed by regulations and standards.

* **Compliance:** Adhering to established guidelines, specifications, or laws.
* **Risk Management:** The process of identifying, assessing, and mitigating risks.
* **NIST** (**National Institute of Standards and Technology**): A federal agency that sets cybersecurity standards for U.S. federal agencies.
* **ISO 27001:** An international standard for managing information security.
* **Framework:** A set of standards, guidelines, and best practices used to manage cybersecurity risk.
* **Policy:** A set of rules that defines what is permitted and what is prohibited within an organization's network.
* **Governance:** The process of ensuring that an organization's security policies and procedures are followed.
* **Audit:** A review of an organization's security policies, procedures, and controls.
* **Regulation:** A rule or directive made and maintained by an authority.

### **Additional Foundational Terms**
* **Cryptojacking:** The unauthorized use of a computer to mine cryptocurrency.
* **Doxxing:** The process of gathering and publishing personally identifiable information about an individual.
* **Pharming:** A type of cyberattack that redirects users from a legitimate website to a fraudulent one.
* **Smishing:** A type of phishing attack that uses text messages.
* **Vishing:** A type of phishing attack that uses voice messages or phone calls.
* **Whaling:** A specific type of phishing attack that targets high-profile individuals.
* **Deep Web:** Parts of the internet not indexed by search engines, including private networks and databases.
* **Dark Web:** A small part of the deep web accessible only with special software.
* **SSL/TLS** (**Secure Sockets Layer/Transport Layer Security**): Protocols that encrypt data transmitted between a web browser and a server.
* **Digital Signature:** A mathematical scheme for demonstrating the authenticity of digital messages or documents.
* **Non-repudiation:** Ensures that a party to a transaction or action cannot later deny having been involved.
* **Access Control List (ACL):** A list of permissions associated with a system resource.
* **Kerberos:** A network authentication protocol that provides strong authentication for applications.
* **Operating System (OS):** The system software that manages computer hardware and software resources.
* **Firmware:** Software programmed onto a hardware device.
* **Cyber Warfare:** Nation-state sponsored cyberattacks targeting other nations.
* **BYOD** (**Bring Your Own Device**): A policy that allows employees to use their personal devices for work.
